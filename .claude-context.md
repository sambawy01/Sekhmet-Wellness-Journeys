## Key Architecture

### Routing
- Uses **HashRouter** (`/#/path`) for GitHub Pages compatibility
- Admin routes: `/#/admin/login`, `/#/admin/dashboard`

### CRM System
- Lead capture forms → Supabase `leads` table
- RLS policies: SELECT for authenticated, INSERT for anon
- Email notifications via Supabase Edge Function + Resend API
- Admin dashboard with lead management

### File Structure (Key Files)
```
src/
  app/
    components/
      Navigation.tsx          — Main navbar with mega menu
      Footer.tsx              — Site footer
      ExitIntentPopup.tsx     — Exit intent popup
      admin/
        AdminLogin.tsx        — CRM login page
        AdminDashboard.tsx    — CRM dashboard
      dental/
```
